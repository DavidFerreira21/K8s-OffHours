# Contribuicao

Obrigado por contribuir com o K8s OffHours.

## Como contribuir

1. Faca um fork e crie uma branch a partir de `main`.
2. Faca as alteracoes.
3. Rode os testes:

```bash
pytest
```

4. Abra um Pull Request com resumo da mudanca.

## Regras basicas

- Mantenha PRs pequenos e objetivos.
- Atualize a documentacao quando necessario.
- Nao inclua secrets/tokens no repositorio.
- Em caso de problema de seguranca, siga `SECURITY.md`.
